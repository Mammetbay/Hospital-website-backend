A backend REST API for a hospital management system, 
built with Node.js, Express.js, and MongoDB.

The project follows a clean MVC architecture:
- controllers/ — request handling logic
- models/       — MongoDB schemas (Mongoose)
- routes/       — API endpoint definitions
- middleware/   — auth & validation layers
- utils/        — shared helper functions

Built as a personal open-source project to practice 
real-world API design patterns.
